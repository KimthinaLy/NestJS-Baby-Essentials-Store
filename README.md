## 🚧 Project Status: Work in Progress
This project is currently under development as part of the **COS2204** course. 
While the core architecture is in place, several features are still being implemented and improved.

## Description
A full-stack e-commerce platform for baby products built for **COS2204: Web Programming** course. This application features a staff management dashboard, product inventory CRUD, and secure session-based authentication.

## Technologies
- **Backend:** NestJS, TypeORM, MySQL
- **Frontend:** EJS, Alpine.js, Tailwind CSS, PrimeIcons
- **Security:** Passport.js, Express-Session, dotenv

## 🚀 Upcoming Features (Technical Roadmap)

To further enhance the **Baby Essential Store**, the following industry-standard concepts will be implemented:

### 🔒 Security & Performance
- [ ] **JWT Authentication:** Moving toward stateless authentication for API scalability.

### 🛒 E-commerce Logic
- [ ] **Cookie-based Cart:** Allowing guest users to persist their shopping cart without being logged in.
